Detailed Description

Lexical analysis routines for MIPS assembly syntax.

MIPS assembly syntax.

Author
François Portet franc.nosp@m.ois..nosp@m.porte.nosp@m.t@im.nosp@m.ag.fr These routines perform the analysis of the lexeme of an assembly source code file.
François Portet franc.nosp@m.ois..nosp@m.porte.nosp@m.t@im.nosp@m.ag.fr These routines perform the analysis of the syntactic aspect of the assembly. 1/ Analysis of the symbol and loading of instruction information from a dictionnaire. 2/ Analysis of instruction type : 3 type of instruction. The aim is to recognize the shape of the instruction using the collection of lexemes. R-Instruction $rd, $rs, $rt, sa I-Instruction $rt, $rs, immediate J-Instruction target 3/ If the intruction is legal, translate to binary. If not, raise an error.
